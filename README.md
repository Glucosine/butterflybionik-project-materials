# butterflybionik-project-materials
仿生昆虫飞行器项目资料库：BP、折页、项目展示与视频索引。

## 项目简介

本项目聚焦仿生昆虫扑翼飞行器的研发与场景化应用，围绕仿生蝴蝶、仿生蜻蜓等多种构型，探索展示服务、科创教育与科研验证。

## 当前资料

- [BP：仿生昆虫飞行器（PDF）](bp/bp_v3.pdf)

## 计划整理

- `bp/`：商业计划书与项目介绍
- `foldout/`：展示折页与海报
- `media/`：视频合集与观看链接
- `docs/`：课程、结构说明与公开技术资料

## 使用说明

本仓库用于项目展示、教育产品介绍与合作交流。正式公开材料中的联系方式、合作方标识和现场影像将按公开授权范围整理。
